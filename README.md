# ANOTHER-HOME-UNITY
The PRE-Build Version of Another Home

## Note on Assets
This project uses the "Interior of International Space Station" asset from the Unity Asset Store:
https://assetstore.unity.com/packages/3d/environments/interior-of-international-space-station-100255

Due to licensing restrictions, this asset is **not included** in the repository.
You must purchase and import it yourself from the Unity Asset Store for the project to run correctly.
